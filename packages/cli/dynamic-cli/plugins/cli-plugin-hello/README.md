## Quick Started
You need to install `@sugarat/cli` first
```sh
npm i -g @sugarat/cli
```

then you will get `q` command

install cli plugin

```sh
q install hello [words...]
q install hello
q install hello 真不戳

# equal
q install @sugarat/cli-plugin-hello@latest
```

then you `q` have hello command
```sh
q hello
```
