## Quick Started
You need to install `@sugarat/cli` first
```sh
npm i -g @sugarat/cli
```

then you will get `q` command

install cli plugin

```sh
q install ep

# equal
q install @sugarat/cli-plugin-ep@latest
```

then you `q` have ep command
```sh
q ep
```
